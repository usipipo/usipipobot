# Fonts Directory

This directory should contain the following font files for the cyberpunk theme:

- `JetBrainsMono-Bold.ttf` - Primary monospace font for headings
- `JetBrainsMono-Regular.ttf` - Primary monospace font for body text

## Download

Fonts can be downloaded from: https://www.jetbrains.com/lp/mono/
